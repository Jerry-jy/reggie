# Reggie外卖项目

## 1、外卖项目介绍

### 项目介绍

Reggie外卖项目是基于SpringBoot为后台框架开发的，整个项目的前端展示页面均为提供好的，只需要编写后台服务接口


项目分为**后台系统管理**和**移动端**两部分

后台系统管理供商家：对菜品、套餐、订单等进行管理维护等功能

移动端供消费者：在线浏览，添加购物车，下单 等功能
